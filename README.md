# 📚 AI Study Companion

An intelligent academic assistant that transforms documents into interactive learning materials using AI.

## ✨ Features

- 📄 **Document Processing** - Upload PDF, PPT, or Images
- 📝 **Smart Summaries** - Get concise summaries
- 📖 **Detailed Overviews** - In-depth topic explanations  
- ❓ **Auto Quiz Generation** - Create MCQs automatically
- 📋 **Short Questions** - Practice questions with answers
- ⭐ **Key Points Extraction** - Important takeaways
- 📚 **Vocabulary Builder** - Learn difficult terms
- 💾 **Save Documents** - Download or save to project
